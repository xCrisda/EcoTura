# 🌱 EcoTura

**EcoTura** es una aplicación web desarrollada como proyecto universitario que busca promover la gestión inteligente de residuos sólidos en la ciudad de Buenaventura.

La plataforma permite a los ciudadanos conocer las rutas de recolección, reportar puntos con acumulación de basura y aprender sobre la correcta separación de residuos para contribuir al cuidado del medio ambiente.

---

## 📋 Características

- 🗺️ Visualización de rutas de recolección.
- ♻️ Guía para la correcta separación de residuos.
- 📍 Reporte de puntos con acumulación de basura.
- 🚛 Simulación de rutas de recolección.
- 📢 Centro de alertas ambientales.
- 🔐 Inicio de sesión para usuarios.

---

## 🚀 Cómo ejecutar el proyecto

1. Clonar el repositorio:

```bash
git clone https://github.com/xCrisda/EcoTura.git
```

2. Abrir la carpeta del proyecto.

3. Ejecutar el archivo **index.html** utilizando **Live Server** en Visual Studio Code.
